# Intro To Web Applications Course

### :pushpin: Project Description
This repository documents the progress through the **Web Applications** course. It includes a series of laboratory exercises covering full-stack web development concepts and a final capstone project.

**Key Features:**
- :white_check_mark: Progressive learning path from HTML/CSS to React.
- :white_check_mark: **Training Plan Generator** - A fully responsive web application.

### :file_folder: Repository Structure

#### :trophy: Final Project: Training Plan Generator (`strona_plany_treningowe`)
A responsive website designed for generating workout plans.
- :page_facing_up: **`index.html`** – Homepage
- :page_facing_up: **`galeria.html`** – Gallery
- :page_facing_up: **`plany.html`** – Workout plan generator
- :page_facing_up: **`kontakt.html`** – Contact page
- :scroll: **`cwiczenia.js`** – Database of exercises
- :scroll: **`script.js`** – Logic for plan generation
- :art: **`style.css`** – Custom styling

#### :books: Laboratories
- :open_file_folder: **`lab_01`** – **HTML & CSS Fundamentals**
  - Introduction to web page structure and styling.
- :open_file_folder: **`lab_02`** – **JavaScript Essentials**
  - Fundamental JavaScript concepts and DOM manipulation.
- :open_file_folder: **`lab_03`** – **Interactive Applications**
  - Game logic, assets integration, and advanced DOM events.
- :open_file_folder: **`lab_04`** – **Backend Development**
  - Python-based backend logic and API implementation (`bookstore-api`).
- :open_file_folder: **`lab_05`** – **React Framework**
  - Modern frontend development using React (`my-app`).
